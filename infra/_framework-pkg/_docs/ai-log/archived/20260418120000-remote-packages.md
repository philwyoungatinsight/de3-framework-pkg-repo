## Design
- GUI is the config editor (opens files in file viewer); `pkg-mgr` owns all filesystem operations
- Symlinks at `infra/<pkg>` pointing into `_ext_packages/<repo>/infra/<pkg>` are committed to git
- `pkg-mgr sync` reconciles filesystem with current YAML config state
- Collision detection: unit-level (`terragrunt.hcl` path overlap) is a hard error; config-level is a warning
