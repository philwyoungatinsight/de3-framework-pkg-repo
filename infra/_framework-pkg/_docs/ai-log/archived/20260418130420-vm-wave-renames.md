# Wave Renames — vm.proxmox.* → vms.proxmox.*

## Summary

Renamed 6 VM-related waves from the legacy `vm.proxmox.*` prefix to a consistent
`vms.proxmox.<category>.<name>` scheme. The new names make the wave's role (utility
VM, web-downloaded image, custom Packer/Kairos image) legible from the name alone.
Also included the previously uncommitted `maas.servers` → `maas.servers.all` rename.
