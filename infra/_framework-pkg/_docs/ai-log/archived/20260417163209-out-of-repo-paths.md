# Eliminate remaining out-of-repo runtime paths

## What changed

Removed three paths that wrote or read outside the git repo:

1. **`utilities/python/validate-config.py`**: flag file path changed from `~/.cache/pwy-home-lab/validate-config-last-run` (user-specific home dir) to `<repo-root>/config/tmp/validate-config-last-run`. Now isolated per checkout.

2. **`config/framework.yaml`**: removed now-unused `flag_file` key under `validate_config:`.

3. **`infra/de3-gui-pkg/.../homelab_gui.py`** (two changes):
   - `_STACK_DIR` default replaced with a `_find_repo_root()` function that walks up from `__file__` looking for `root.hcl` as the repo-root sentinel. Uses `root.hcl` rather than `set_env.sh` because the GUI app directory contains its own `set_env.sh`.
   - `gui_preview.html` path changed from hardcoded `/tmp/gui_preview.html` to `Path(os.environ.get("_GUI_DIR", "/tmp")) / "gui_preview.html"`.
