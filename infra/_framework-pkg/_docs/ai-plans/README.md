# Goal
- Create a place for ai to record its plans
- This should be part of a skill that will:
  - Create a plan
  - Review the code and docs to make sure they are correct
  - Stop and Ask the user for input on any required decisions or input
  - Proceed if there are no questions
  - Clear the context (/clear)
  - Execute the plan
  - Comment and Commit the code (/ship)

# Motivation
- It is useful to be able to clear the context before coding 
  in order to avoid compactions during coding.
