# Plan: Add fw-repo-mgr — Framework Repository Manager

## Objective

Add a new framework tool `fw-repo-mgr` that builds and maintains downstream framework
repos. Config declares a registry of **source repos** (templates to clone from) and a list
of **target repos** to build. Each target specifies which source to clone, which packages
to include, and optionally an upstream remote to push to.

This supports two complementary workflows without any special-casing:

- **Splitting**: clone a multi-package source, keep only one package → publish single-package repos
- **Combining**: clone a template source, import packages from multiple single-package repos via `repo`/`source` entries → publish an app-repo

`fw-repo-mgr -b [name]` = clone source + prune unwanted packages from `infra/` + overwrite
`framework_packages.yaml` + `pkg-mgr sync` + commit + optional push.

---

## Context

### Core design

A framework repo is defined by its `framework_packages.yaml`. Everything else comes from
the cloned source template. fw-repo-mgr therefore only needs to:

1. Resolve the source repo URL (from `source_repos` registry or inline `source_url`)
2. Clone/pull to `$HOME/<framework_repo_dir>/<name>/`
3. **Prune** real directories from `infra/` that are NOT in the target `framework_packages`
   list (symlinks are left for `pkg-mgr sync` to handle)
4. Overwrite `infra/_framework-pkg/_config/framework_packages.yaml` with the target spec
5. Run `pkg-mgr sync` inside the target (clones imported packages, creates/removes symlinks)
6. Commit + push to `upstream_url` if configured

### Why a prune step is needed

When the source template has 12 packages and the target only wants 2, the 10 unwanted
real directories will be committed to the target repo unless explicitly removed. Symlinks
(imported packages) are already managed by `pkg-mgr sync --clean`, but real directories
must be pruned by fw-repo-mgr before sync runs.

Rule: delete `infra/<pkg>/` in the target if it is a **real directory** (not a symlink)
AND `<pkg>` does not appear in the target's `framework_packages` list.

### Patterns followed

- **pkg-mgr** (`_framework/_pkg-mgr/run`): bash, inline `python3 -c` for YAML, colored
  table output. fw-repo-mgr matches this style exactly.
- **`framework_package_repositories.yaml`**: the `source_repos` section mirrors this —
  a named registry of repos so targets can reference `source_repo: de3-runner` instead of
  repeating URLs.
- **`framework_packages.yaml` schema**: the `framework_packages` entries inside each
  `framework_repo` use the exact same schema as `framework_packages.yaml` (`name`, `public`,
  `repo`, `source`, `git_ref`, `import_path`, `config_source`). fw-repo-mgr writes this
  section verbatim into the target's `framework_packages.yaml`.

### `infra/_framework-pkg/_config/framework_manager.yaml` — create

```yaml
framework_manager:

  # Base directory under $HOME where target repos are built.
  # 'git' → $HOME/git/<name>/   '.' → $HOME/<name>/
  framework_repo_dir: 'git'

  # Registry of known source/template repos (resolved by name in framework_repos).
  # Mirrors the role of framework_package_repositories.yaml for pkg-mgr.
  source_repos:
    - name: de3-runner
      url: https://github.com/philwyoungatinsight/de3-runner.git
      ref: main          # default ref when not overridden per target

  # Target repos to build.
  framework_repos:

    # Example A: single-package repo (splitting de3-runner → proxmox-pkg-repo)
    - name: proxmox-pkg-repo
      source_repo: de3-runner          # resolved via source_repos registry above
      # source_url / source_ref can be used instead of source_repo for one-off URLs
      upstream_url: https://github.com/philwyoungatinsight/proxmox-pkg-repo.git  # optional
      upstream_branch: main            # optional; default: main
      framework_packages:
        - name: _framework-pkg
          public: true
        - name: proxmox-pkg
          public: true
        - name: proxmox-pkg-repo       # deployment package
          public: false

    # Example B: app-repo importing from individual package repos (combining)
    - name: my-homelab
      source_repo: de3-runner
      upstream_url: https://github.com/philwyoungatinsight/my-homelab.git
      framework_packages:
        - name: _framework-pkg
          public: true
        - name: proxmox-pkg
          public: true
          repo: proxmox-pkg-repo
          source: https://github.com/philwyoungatinsight/proxmox-pkg-repo.git
          git_ref: main
          import_path: proxmox-pkg
          config_source: my-homelab
        - name: my-homelab
          public: false
```

Notes:
- `source_repo`: name from `source_repos` list; resolved to URL + ref at build time.
- `source_url` / `source_ref`: used directly when no `source_repos` entry exists for
  the source. Either `source_repo` or `source_url` must be present.
- `upstream_url`: if absent, local build only (no push).
- `upstream_branch`: defaults to `main`.

**Config readers:**

```bash
_fw_repo_base_dir() {
  python3 -c "
import yaml, pathlib, os
cfg = pathlib.Path('$FW_MGR_CFG')
if not cfg.exists(): print(''); exit(0)
d = yaml.safe_load(cfg.read_text()) or {}
rel = d.get('framework_manager', {}).get('framework_repo_dir', '')
if not rel: print(''); exit(0)
home = os.path.expanduser('~')
print(home if rel == '.' else os.path.normpath(os.path.join(home, rel)))
" 2>/dev/null || echo ""
}

# Returns JSON array of framework_repo entries
_list_repos() {
  python3 -c "
import yaml, pathlib, json
cfg = pathlib.Path('$FW_MGR_CFG')
if not cfg.exists(): print('[]'); exit(0)
d = yaml.safe_load(cfg.read_text()) or {}
print(json.dumps(d.get('framework_manager', {}).get('framework_repos', [])))
"
}

# Resolve source URL + ref for a target repo name.
# Checks source_repo (registry lookup) then falls back to source_url/source_ref.
_resolve_source() {   # _resolve_source <repo_name>  →  prints "<url> <ref>"
  python3 -c "
import yaml, pathlib, json, sys
cfg = pathlib.Path('$FW_MGR_CFG')
d = yaml.safe_load(cfg.read_text()) or {}
fm = d.get('framework_manager', {})
source_registry = {s['name']: s for s in fm.get('source_repos', [])}
repos = fm.get('framework_repos', [])
for r in repos:
    if r.get('name') != '$1': continue
    sname = r.get('source_repo', '')
    if sname:
        s = source_registry.get(sname, {})
        url = s.get('url', '')
        ref = r.get('source_ref') or s.get('ref', 'main')
    else:
        url = r.get('source_url', '')
        ref = r.get('source_ref', 'main')
    print(url + ' ' + ref)
    sys.exit(0)
print(' ')
"
}

# Read a scalar field from a named framework_repo entry
_repo_field() {   # _repo_field <name> <field>
  python3 -c "
import yaml, pathlib
cfg = pathlib.Path('$FW_MGR_CFG')
d = yaml.safe_load(cfg.read_text()) or {}
for r in d.get('framework_manager', {}).get('framework_repos', []):
    if r.get('name') == '$1':
        print(r.get('$2', '')); exit(0)
print('')
"
}
```

**`_prune_infra` helper:**

```bash
# Remove real directories from infra/ in the target that are NOT in framework_packages.
# Symlinks (imported packages managed by pkg-mgr) are left untouched.
_prune_infra() {
  local repo_dir="$1" repo_name="$2"
  local keep_json
  keep_json=$(python3 -c "
import yaml, pathlib, json
cfg = pathlib.Path('$FW_MGR_CFG')
d = yaml.safe_load(cfg.read_text()) or {}
for r in d.get('framework_manager', {}).get('framework_repos', []):
    if r.get('name') == '$repo_name':
        print(json.dumps([p['name'] for p in r.get('framework_packages', [])]))
        exit(0)
print('[]')
")
  python3 -c "
import json, pathlib, shutil, sys
keep = set(json.loads('$keep_json'))
infra = pathlib.Path('$repo_dir/infra')
if not infra.exists(): sys.exit(0)
for entry in infra.iterdir():
    if entry.is_symlink(): continue          # pkg-mgr manages these
    if not entry.is_dir(): continue
    if entry.name not in keep:
        print(f'  prune: infra/{entry.name}')
        shutil.rmtree(entry)
"
}
```

**`_build_repo` function:**

```bash
_build_repo() {
  local repo_name="$1" force_push="${2:-false}"

  local repo_base; repo_base=$(_fw_repo_base_dir)
  [[ -z "$repo_base" ]] && { echo "ERROR: framework_repo_dir not set in $FW_MGR_CFG" >&2; exit 1; }

  local repo_dir="$repo_base/$repo_name"

  # Resolve source
  local source_info; source_info=$(_resolve_source "$repo_name")
  local source_url="${source_info%% *}"
  local source_ref="${source_info##* }"
  [[ -z "$source_url" ]] && { echo "ERROR: no source_repo or source_url for '$repo_name'" >&2; exit 1; }

  local upstream_url;    upstream_url=$(_repo_field "$repo_name" upstream_url)
  local upstream_branch; upstream_branch=$(_repo_field "$repo_name" upstream_branch)
  upstream_branch="${upstream_branch:-main}"

  # Step 1: clone or update
  if [[ ! -d "$repo_dir/.git" ]]; then
    echo "Cloning $source_url ($source_ref) → $repo_dir ..."
    git clone "$source_url" "$repo_dir"
    git -C "$repo_dir" checkout "$source_ref"
  else
    echo "Updating $repo_dir ..."
    git -C "$repo_dir" fetch origin
    git -C "$repo_dir" checkout "$source_ref"
    git -C "$repo_dir" pull --ff-only origin "$source_ref" 2>/dev/null || true
  fi

  # Step 2: prune infra/ — remove real dirs not in framework_packages
  echo "Pruning infra/ to target package list ..."
  _prune_infra "$repo_dir" "$repo_name"

  # Step 3: overwrite framework_packages.yaml
  local fw_pkgs_out="$repo_dir/infra/_framework-pkg/_config/framework_packages.yaml"
  mkdir -p "$(dirname "$fw_pkgs_out")"
  _write_framework_packages_yaml "$repo_name" "$fw_pkgs_out"
  echo "Wrote: infra/_framework-pkg/_config/framework_packages.yaml"

  # Step 4: pkg-mgr sync inside target
  echo "Running pkg-mgr sync in $repo_dir ..."
  (cd "$repo_dir" && source set_env.sh && "$PKG_MGR" sync)

  # Step 5: commit
  git -C "$repo_dir" add -A
  if ! git -C "$repo_dir" diff --cached --quiet; then
    git -C "$repo_dir" commit -m "chore(fw-repo-mgr): update packages for $repo_name"
  else
    echo "Nothing to commit in $repo_dir"
  fi

  # Step 6: push if upstream_url set
  if [[ -n "$upstream_url" ]]; then
    git -C "$repo_dir" remote get-url origin &>/dev/null || \
      git -C "$repo_dir" remote add origin "$upstream_url"
    local push_args="-u origin $upstream_branch"
    [[ "$force_push" == "true" ]] && push_args="--force $push_args"
    git -C "$repo_dir" push $push_args
    echo "Pushed: $repo_name → $upstream_url ($upstream_branch)"
  fi

  echo "Done: $repo_name → $repo_dir"
}
```

**`_write_framework_packages_yaml` helper:**

```bash
_write_framework_packages_yaml() {
  local repo_name="$1" out_path="$2"
  python3 - "$repo_name" "$out_path" "$FW_MGR_CFG" <<'PYEOF'
import sys, yaml, pathlib
repo_name, out_path, cfg_path = sys.argv[1], sys.argv[2], sys.argv[3]
d = yaml.safe_load(pathlib.Path(cfg_path).read_text()) or {}
repos = d.get('framework_manager', {}).get('framework_repos', [])
pkgs = next((r.get('framework_packages', []) for r in repos if r.get('name') == repo_name), [])
out = {'framework_packages': pkgs}
pathlib.Path(out_path).write_text(yaml.dump(out, default_flow_style=False, sort_keys=False))
PYEOF
}
```

**`status` sub-command:**

```bash
_status() {
  local repos_json; repos_json=$(_list_repos)
  local repo_base; repo_base=$(_fw_repo_base_dir)

  printf "%-22s %-30s %-6s %-25s %s\n" "NAME" "LOCAL PATH" "PKGS" "UPSTREAM" "STATUS"
  printf "%-22s %-30s %-6s %-25s %s\n" "----" "----------" "----" "--------" "------"

  echo "$repos_json" | python3 -c "
import json, sys
for r in json.load(sys.stdin):
    pkgs = len(r.get('framework_packages', []))
    up = r.get('upstream_url', '(none)')
    # shorten github URLs for display
    up = up.replace('https://github.com/', 'github.com/')
    print(f\"{r['name']}|{pkgs}|{up}\")
" | while IFS='|' read -r name pkg_count upstream; do
    local repo_dir="$repo_base/$name"
    local status
    if [[ ! -d "$repo_dir/.git" ]]; then
      status="! not cloned"
    elif ! git -C "$repo_dir" diff --quiet 2>/dev/null || \
         ! git -C "$repo_dir" diff --cached --quiet 2>/dev/null; then
      status="? dirty"
    else
      status="✓ clean"
    fi
    local short_path="${repo_dir/$HOME/~}"
    printf "%-22s %-30s %-6s %-25s %s\n" "$name" "$short_path" "$pkg_count" "$upstream" "$status"
  done
}
```

**Main dispatch:**

```bash
usage() {
  cat <<EOF
fw-repo-mgr — framework repository manager

Usage:
  fw-repo-mgr -b [<name>] [--force-push]    Build/sync all or named target repo
  fw-repo-mgr build [<name>] [--force-push]  (same as -b)
  fw-repo-mgr status                         Show state of all configured target repos
  fw-repo-mgr help                           Show this message

Workflow:
  1. Add entries to framework_manager.yaml (source_repos + framework_repos)
  2. fw-repo-mgr -b <name>   → clone source, prune, update packages, sync, commit, push
EOF
}

CMD="" FORCE_PUSH=false REPO_NAME=""

while [[ $# -gt 0 ]]; do
  case "$1" in
    -b|build)       CMD="build"; shift ;;
    status)         CMD="status"; shift ;;
    help|--help|-h) CMD="help"; shift ;;
    --force-push)   FORCE_PUSH=true; shift ;;
    -*)             echo "ERROR: unknown flag $1" >&2; exit 1 ;;
    *)              REPO_NAME="$1"; shift ;;
  esac
done

case "${CMD:-help}" in
  build)
    if [[ -n "$REPO_NAME" ]]; then
      _build_repo "$REPO_NAME" "$FORCE_PUSH"
    else
      _list_repos \
        | python3 -c "import json,sys; [print(r['name']) for r in json.load(sys.stdin)]" \
        | while IFS= read -r name; do _build_repo "$name" "$FORCE_PUSH"; done
    fi
    ;;
  status) _status ;;
  help)   usage ;;
  *)      usage; exit 1 ;;
esac
```
